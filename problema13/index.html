Problema 13

Pseudo código

Realice un algoritmo para leer las calificaciones de N alumnos y determine el número de aprobados y reprobados.

Solución

1. Ingresar el número de alumnos

2. Solicitar las notas de cada alumno y almacenarlas

3. Analizar cada una de las notas con la ayuda de un bucle. Previamente al bucle se crearan las variables aprobados y desaprobados. A cada nota se le aplican las condiciones:

  • Si su nota es mayor a 12, entonces el alumnos esta aprobado, y se agregará 1 al numero de aprobados
  • De no ser así se agregará 1 al numero de desaprobados.

4. Finalizando el bucle, mostraremos el numero de aprobados y desaprobados obtenidos.
